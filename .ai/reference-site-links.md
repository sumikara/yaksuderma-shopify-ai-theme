This file contains real external reference URLs for the YaksuDerma Shopify redesign.

## 2. Important Rules

External sites are for inspiration only.

Codex must not copy:

- Exact layouts
- Exact text
- Exact images
- Exact code
- Logos
- Product claims
- Brand identity
- Regulatory claims
- Certificate claims
- Medical claims
- Pricing strategy
- Before / after claims
- Proprietary design systems

Codex should only extract general patterns such as:

- Product grid clarity
- Navigation hierarchy
- Mobile menu behavior
- Homepage section flow
- Product card spacing
- Product page information hierarchy
- Trust / authenticity placement
- Customer support placement
- Clean color usage
- Professional catalog organization
