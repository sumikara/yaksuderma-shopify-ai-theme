## 2. Important Rule

External references are for inspiration only.

Codex must not copy:

- Exact layouts
- Exact text
- Exact images
- Exact code
- Brand identity
- Product claims
- Proprietary design systems
- Logos
- Icons
- Photography
- Page structure one-to-one

Codex should extract general UX and visual principles only.

Good use:

```text
Use this reference to understand clean product hierarchy, spacing, mobile grid behavior, or trust-section placement.
```

Bad use:

```text
Copy this homepage exactly.
```

## 8. YaksuDerma-Specific Reference Needs

Codex should look for or infer inspiration around these UI needs:

### Homepage

Needs:

- Clear hero
- Trust/authenticity bar
- Category showcase
- Featured collections
- Support block
- Optional wholesale inquiry
- Clean footer

Avoid:

- Single-banner-only homepage
- Too many promotional blocks
- Heavy discount-first layout

### Header

Needs visible navigation:

```text
Home
Toxins
Fillers
Skin Boosters
Fat Dissolvers
Collagen Stimulators
Accessories
About
```

Avoid:

- Generic “Shop” replacing the agreed structure
- Too many top-level labels beyond the final menu
- Crowded desktop header
- Hidden category access on mobile

### Product Cards

Needs:

- Clean product image
- Long product title support
- Dynamic Shopify price
- Optional subtle badge
- Mobile readability
- No marketplace look

Avoid:

- Red-heavy sale styling
- Tiny text
- Overcrowded badges
- Hardcoded prices

### Product Page

Needs:

- Product image clarity
- Dynamic price
- Product form preserved
- Variant logic preserved
- Product details accordions
- Authenticity block
- Shipping/availability note
- Support CTA
- Related products
- Product family comparison table when useful

Avoid:

- Breaking product form
- Medical claims
- Hardcoded family tables
- Fake before/after claims

### Collection Page

Needs:

- Clean collection header
- Strong product grid
- Filtering and sorting preserved
- Long names readable
- Mobile 2-column grid if readable
- Clear category title mapping

Avoid:

- Removing filters
- Crowded grid
- Hidden pricing
- Category names that hurt SEO
